# The ```Ring``` Programming Language


## ```Ring```语言的功能

- 需要方便的组织各种依赖、并发编译、链接
- 静态类型 (Static Typing) 语言
- 执行方式：解释执行、虚拟机执行
- 支持垃圾回收GC
- 支持面向对象
- import 导包
- 支持bind 绑定函数
- auto类型
- lambda表达式
- 支持闭包
- 支持 多个 defer
