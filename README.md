## 1. What is Ring Language

**Ring** 是由GeneralSandman与2021年开始设计实现的高级编程语言，目标：轻量级脚本编程语言、解决目前现有编程语言的痛点。


- 静态类型系统

- 支持**面向过程式**、**面向对象**

- 简洁、灵活的语法规范

- 源代码编译生成字节码由**Ring虚拟机**执行

- 较为丰富的标准库

- 可原生调用C语言

- 跨平台运行

- 基于**C++11**设计实现



## 2. Get Ring Features

- 基础数据类型: bool, int, double, string
- 派生数据类型: array, class
- 数学运算
- 基本控制语句: if, else, for, for-range, do-for, break, continue
- 全局变量, 局部变量
- package包命名机制
- 动态内存管理&垃圾回收机制
