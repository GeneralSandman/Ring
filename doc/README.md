# The ```Ring``` Programming Language


## What is ```Ring```

```Ring```是一个轻量级的编程语言，支持**过程式编程（Procedural programming）**，**面向对象**。

## ```Ring```编程语言语言的功能

- 静态类型 (Static Typing) 语言
- 支持垃圾回收GC
- 执行方式：解释执行、虚拟机执行
- 支持面向对象
- 需要方便的组织各种依赖、并发编译、链接
- import 导包
- 支持函数闭包、匿名函数与类型推导
- 支持 defer
